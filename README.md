## 題目說明
### 規則
- 每一個 cell 絕對不能重疊
- 一格一格的東西叫做 pin，每個 pin 被 cell 所覆蓋的面積不能超過 BinMaxUtil （每一個 pin 都有一樣的 BinMaxUtil），不然的話 D 會 +1，權重是 lambda
